翻译页面标题:

如果你的应用程序是多语言的，你也需要翻译页面的标题。在这一段中，我将告诉你如何做到这一点:

为了翻译这个应用程序，我将使用 ngx-translate 库。

npm install @ngx-translate/core @ngx-translate/http-loader

我将把应用程序翻译成3种语言，所以我将创建3个翻译文件；每种语言一个:

如何使用ngx-translate lib我就不赘述了，如果你不熟悉，我推荐你这篇很棒的文章:Angular+@ ngx-translate+Typings作者卡洛斯·卡瓦列罗。

要翻译页面标题，我们必须覆盖页面标题的角度行为(title strategy)；通过创建一个CustomPageTitleStrategy类，该类将获取标题翻译关键字并设置正确的翻译标题。

要覆盖TitleStrategy，我们只需将它添加到app.module providers:

{ provide: TitleStrategy, useClass: CustomPageTitleStrategy }
