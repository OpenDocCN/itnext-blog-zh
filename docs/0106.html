Java是大数据项目中使用最广泛的编程语言之一，它的流行部分归功于其广泛的生态系统。Java编程提供了对这个由几个库、框架和API组成的生态系统的访问。

在一系列文章中，我将简要介绍大数据项目中最常用的Java库、框架和API。上一篇文章专门介绍了九个主要的机器学习库，它们对不同项目的开发者都有帮助。

本文致力于Java库，这些库解决了自然语言处理(NLP)中的常见问题，NLP是机器学习的一个子领域。

Java中最常用的5个自然语言处理库

1.斯坦福科伦普

一套GPL许可的语言分析工具，用于处理文本。它包含用于标记化、情感分析、词性标注(POS)、解析、命名实体识别(NER)、指代消解等的工具。

事实上，斯坦福CoreNLP为不同的斯坦福NLP工具提供了具体的API。这些工具包括:分词器、词性标注器、命名实体识别器、解析器、指代消解系统、情感分析、引导模式学习、开放信息抽取。斯坦福NLP小组也有其他的 NLP软件，它们是开源的，可以在GNU通用公共许可证下获得。

2. Apache OpenNLP

一套用于处理自然语言文本的工具。它提供了类和接口来执行各种自然语言处理任务，例如标记化、句子分割、词性标注、命名实体提取、组块、解析，以及共指消解。

OpenNLP旨在填补ASF在人类语言处理工具方面的空白。它提供了一个坚实的库，包括最大熵和基于机器学习的感知器。

3.灵管

用Java编写的计算语言学工具包。它包括最常见的自然语言处理任务的方法，如标记化，句子检测，命名实体检测，共指消解，分类，聚类，词性标注，一般组块，模糊字典匹配。

除了核心API，LingPipe还提供了一系列预编译的模型。这些模型基于每种语言、体裁和训练语料库的任务。LingPipe在运行时加载这些模型，它们向核心LingPipe API提供特定的细节。

值得注意的是，Lingpipe文档丰富，包含许多教程，对任何NLP初学者都很有帮助，不管是使用它还是其他工具。

4.浇口嵌入

用于计算语言处理和文本挖掘的开源Java类库(JCL)。Embedded是一组作为jar交付的工具(概述其包和类)。它原本用于Java、基于JVM的语言，如 Groovy ，并通过JNI或其他语言的类似网关使用。它能够在不同的应用程序中实现语言处理功能。

在所有基于Gate的系统中使用的Gate Embedded实际上是GATE(文本工程通用架构)的子项目之一。 GATE 是一套解决NLP问题的工具，包括一个开发环境，一个基于工作流的 web应用，一个 Java库，一个架构和一个流程。

5.语言工具包的机器学习— MALLET

一个用于文本统计自然处理的开源Java工具包。它包含包，用于文档分类、聚类、主题建模、信息提取和文本的多种ML应用。它还包括将文本文档转换成数字表示的例程，以便进行有效的处理。

此外，MALLET还可以添加 GRMM包，这是一个在任意结构的图形模型中进行推理和学习的工具包。
