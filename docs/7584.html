9种语言的文本&支持1D和2D条形码

2022年WWDC，苹果发布 VisionKit 的DataScannerViewController。它是一个无需网络连接即可在本地检测文本和条形码的UIViewController。回答率和准确率都很高。它支持以下所有9种语言:

  1. 英语
  2. 法语
  3. 意大利的
  4. 德国人
  5. 西班牙语
  6. 葡萄牙语
  7. 中文[繁体及简体版本]
  8. 日本人
  9. 韩国的

注意:
但是，使用该功能有一些限制。例如，iOS设备必须拥有 A12或更新的仿生芯片，并运行 iOS 16或更新版本。
参考:苹果官方文档

如果你在应用中使用谷歌云视觉API 或亚马逊认知API ，你可能会通过实现DataScannerViewController节省很多钱。你只需要4个步骤来设置整个功能。让我们一起深入了解最新的技术！

结论

DataScannerViewController是嵌入在VisionKit框架中的免费使用的UIViewController。它通过 Google Cloud Vision API 和 Amazon Rekognition API 提供类似的服务。它可以检测9种不同语言的文本以及1D & 2D条形码。它又快又可靠。

DataScannerViewController的唯一限制是只能在iOS 16+ 上搭载 A12-bionic芯片的设备上运行。
