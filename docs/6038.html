入门指南

装置

运行以下命令，通过yarn或npm安装Variabless包:

npm i -D @ngneat/variabless
yarn add -D @ngneat/variabless

使用

有两种方法可以通过CLI或Webpack插件使用Variabless。

推荐的方法 是使用Webpack插件，因为它为您提供了在开发过程中添加/删除变量的能力，同时您还可以处理项目，并在进行更改时看到结果。
