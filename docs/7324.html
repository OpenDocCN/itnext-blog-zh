忒伊亚作为默认接口

如果你在Killercoda上创建一个场景，那么默认情况下它会激活第一个终端标签。然后，用户可以选择编辑器链接切换到忒伊亚:

这是可以改变的，所以忒伊亚(编辑器标签)是默认选中的，如果你把它添加到你的index.json:

{
  "title": "Theia",
  "backend": {
    "imageid": "ubuntu"
  },
  "interface": {
    "layout": "ide"
  }
}
