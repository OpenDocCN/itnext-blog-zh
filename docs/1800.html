了解如何在Dart中使用这个JS表示库

在今天的教程中，我们将学习如何在Dart web项目中使用fullPage.js演示库。我们将使用Dart团队的 js 包来实现这一点。

设置

安装 stagehand 并生成web项目:

$ pub global activate stagehand
$ mkdir my_project && cd my_project
$ stagehand web-simple

修改 pubspec.yaml 文件的dependencies部分:

dependencies:
  js: ^0.6.0

并更新您的依赖关系:

$ pub get
