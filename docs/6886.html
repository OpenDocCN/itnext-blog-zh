我已经研究本地数据库两个月了，以下是我的浅见。

我将评估4种类型的包

  • Key-Value :只存储基本的原始数据
  • SQL :关系数据库——表格
  • No-SQL :非关系数据库— JSON
  • 以安全为中心:敏感数据存储
