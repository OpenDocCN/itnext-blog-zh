概念证明

为了避免构建所有必要的web3基础设施，我们将使用 Thirdweb SDK ，它为快速构建和部署智能合约提供了API。我们将使用 React 作为框架，使用 Chakra UI 构建布局和UI组件。NFT Instagram将具有以下功能:

  • 连接元掩码加密钱包；
  • 将NFT图像上传到 IPFS
  • 造币厂测试多边形孟买区块链
  • 从区块链网络获取NFT数据
  • 显示NFT图像馈送

yarn create react-app react-nft-insta --template typescript

接下来是安装所有必要的依赖项

第三个web依赖项:

yarn add @thirdweb-dev/react @thirdweb-dev/sdk ethers

然后是查克拉UI依赖和react-icons

yarn add @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^6 react-icons
