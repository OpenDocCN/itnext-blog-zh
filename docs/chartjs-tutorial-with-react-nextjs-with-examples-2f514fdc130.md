## 安装图表

在我们创建了 react 应用程序之后，现在我们可以**安装 **chartjs** 所需的**库，以便在项目中使用它。

npm

```
npm install --save react-chartjs-2 chart.js
```

故事

```
yarn add react-chartjs-2 chart.js
```

安装后，我们可以开始在我们的应用程序中使用 chartjs
