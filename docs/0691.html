用Sage 9和UIkit开发WordPress主题

原文：https://itnext.io/wordpress-theme-development-with-sage-9-and-uikit-10752e1d09d2?source=collection_archive---------4-----------------------#2018-05-08

我作为Wordpress开发者已经工作了一段时间，我不得不说 Sage 是一个很好的WordPress入门主题。入门主题包括Bootstrap 4、Foundation…开箱即用，如果你愿意，你甚至可以集成其他前端框架。

现在，Sage 9包括了网络包！虽然是免费的，但这是多么划算啊！)

今天，我将向你展示如何将 UIkit框架集成到Sage 9中。如果你想使用Sage 8.53，请阅读这篇文章:我最喜欢的自定义主题开发的WordPress启动器

如果你想马上使用，我在Github 上创建了一个 WordPress Starter。您只需要克隆项目并开始工作:)

第一步:你需要安装 WordPress

第二步:进入你的主题文件夹，开始安装Sage主题

composer create-project roots/sage your-theme-name

当它询问您想要使用哪个框架时，记得选择 NONE 。

第三步:安装UIkit

yarn install uikit

第四步:添加UIkit样式表

// resources/assets/styles/main.scss// @import "~some-node-module";
@import "~uikit/dist/css/uikit.css";

步骤4: 添加UiKit脚本和图标

// resources/assets/scripts/main.jsimport UIkit from 'uikit';
import Icons from 'uikit/dist/js/uikit-icons';// loads the Icon plugin
UIkit.use(Icons);

第五步:从现在开始，快乐WordPress开发！

yarn && yarn build

不要忘记在你的项目中使用git)
