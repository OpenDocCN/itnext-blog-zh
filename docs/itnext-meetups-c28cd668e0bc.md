# **知识之咬|** 零售业如何应对瞬息万变的数字世界？

当然，在过去的一年里，一个性能良好的(移动)网站和完全数字化的体验的重要性大大增加了。事实证明，采用全渠道战略的零售商能够更长久地留住 91%的客户，这是完全有道理的。*您如何应对快速变化的数字世界？* *您客户的数字体验已经是您的头等大事了吗？*

**扬声器**

`**Daniel Gebler, CTO of Picnic**`

`**Önder Ceylan, Principle of LINKIT’s JavaScript Squad**`

虚拟旅行|解冻您的数据—云中的现代大数据工程架构

云的采用越来越多，公司正在寻找新的大数据架构，这些架构可以灵活、可靠，并且可以处理各种数据处理模型，以支持各种业务需求。但是，使用公共云并不意味着您只是移动您的内部基础架构，一旦您可以使用新的架构方法来提高可扩展性、可靠性、并发性、服务管理等。在本次演示中，我们将讨论如何在云中实现大数据处理的现代化。

*   云中的 Lambda 架构
*   将数据存储与数据处理分离
*   编排流程
*   使用公共云的优势和劣势
*   管理并发性
*   托管服务

智能边缘|第 1 部分| # DefrostyourData 虚拟公路旅行

现在每天都要收集大量的数据。一些数据是为大规模分析而保留的，而其他数据只有在实时分析时才有价值。当需要实时处理时，边缘计算提供了一种解决方案。它为更接近数据源和需要数据的地方的数据收集和分析提供了理想的解决方案，避免了延迟。与云相结合，它可以为每一项业务需求提供解决方案。在第一部分中，我们将讨论边缘计算背后的理论，以及如何最好地利用它。

机器学习工程银河系漫游指南

机器学习工程星系中有数百万颗行星。在将机器学习(ML)模型投入生产时，很容易迷失方向。尽管互联网提供了许多关于培训和部署简单 ML 模型的信息，但是将 ML 模型集成到真实的公司设置中真的很难，更不用说正确的调试、测试、部署和监控了。

让我通过分享技巧、诀窍和最喜欢的开源工具来指导你完成这个过程，这会让你的生活变得更加容易。因此，最终，您将知道从哪里开始您的部署之旅，使用什么工具，以及问什么问题。

*   传统与人工智能驱动的软件
*   MLOps
*   动手操作 ML 服务管道
*   分批上菜
*   近乎实时的在线服务
*   嵌入式模型的实时服务

非常适合第一次承担 ML 模型部署任务的软件工程师。没有假设 ML 知识。
