创建新项目

在名称ng-ssr下创建一个节点项目——如果需要，您可以更改它。

npx nx g @nrwl/node:application ng-ssr

安装依赖项

yarn add express

添加&服务简单快捷App 

// main.tsimport { http } from '@google-cloud/functions-framework'
import * as express from 'express'export const app = express()app.get('/', (req, res) => {
  res.send('hello world')
})http('api', app)

过程很简单。这就是创建具有云功能的Express服务器(第二代)所需的全部内容

// run these two commands in a separate terminals
yarn watch:ssr
yarn build ng-ssr --watch
