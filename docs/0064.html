三星(Samsung)的新款顶级设备Galaxy Note 7推出不久，第一批熔化智能手机的消息，甚至一辆汽车也出现了在着火的情况。熟悉智能手机市场生产流程的专家怀疑三星设备的问题是由于设备的快速启动造成的。

三星将在推出新手机的过程中，提前制定生产期限，以便在新iPhone 7上记录“相对枯燥”的内容。尽管有许多新功能和电话设计更改，但供应商的截止日期仍将提前。美国消费者产品安全委员会(以下简称“T4”)声明，电话“T5”可能会因电池连接器过热而引发火焰。这可能是因为3.500 mah的(太大)电池容量太小。

三星Galaxy Note 7的销售暂时停止，预计将于9月21日在美国继续。在荷兰，该设备尚未上市。

与此同时，三星已在绳索上实施召回行动，将250万台设备召回工厂。瑞士信贷集团和其他专家估计了大约的召回费用。十亿美元用于三星。此外，该股还受到韩国电子产品制造商的打击和声誉受损。

插入式iPhone 7

从9月16日起，新款iPhone 7将在荷兰上市，因此，苹果公司将推出三星Galaxy Note 7。这个电话似乎也不能避免儿童疾病。数个使用者表示，当设备负载较重时，设备会发出哔声。本装置的喇叭似乎有「T0」的负载，这是一种磁振产生非预期声音的效果。问题还没有得到苹果的承认。
