React Native是一个可靠、切实的解决方案，可以快速开发应用程序，因为90–95%的代码将跨平台共享。

在最近的React Native项目中，我们决定使用Braintree插件UI集成来处理支付，当您使用Braintree时，这是一个轻松的胜利。这是一个由braintree提供的伟大的库(然后由 wgltony 移植到React Native ),以节省开发者构建他们自己的支付集成的需要。按照wgltony的github repo 上的步骤，很容易为React Native进行设置和工作。在React Native上使用这种集成的唯一缺点是很难定制它的UI。为了做到这一点，您必须深入研究本机代码。
