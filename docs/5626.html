因此，如果你没有时间用 React Native 、 Flutter 或者用 Swift 或 Kotlin 构建你的移动应用，PWA仍然是一个不错的低门槛选择。

添加了可屏蔽图标让你的应用程序终于有了一个更吸引人的图标，如果你决定install它，它将会出现在你的手机屏幕上。

如果您想尝试可屏蔽图标，请查看可屏蔽
应用——它允许您轻松地为您的应用创建可屏蔽图标。

什么能让你的应用程序更吸引人，让它看起来更像是native的快捷方式。

快捷方式可以通过在手机上长按和在桌面上右击来实现。当用户长按时，他会看到一个上下文菜单，里面有你的应用程序的某些动作的链接。
